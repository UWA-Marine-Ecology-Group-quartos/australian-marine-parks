# australian-marine-parks

## Steps to use this template for a new marine park:

1.  Copy the `r/TEMPLATE/` folder and save in the `r/` folder as your park name.
2.  Edit `r/TEMPLATE/02_create-report_appendix-A-natural-values/00_config.yml` and `r/TEMPLATE/03_create-report_appendix-B-pressures/00_config.yml` to set your marine park and project name (currently set to "template")
3.  Copy `TEMPLATE/` folders in `data/`, `output/model-output/`, and `plots/` and set to your park name.
4.  Run scripts in number order.
    - Note that the template scripts were duplicated from the Geographe marine park, so they need to be read through and edited to be appropriate for your park.
    - For example:
      - the spatial extent is currently set to Geographe Bay and will need to be adjusted,
      - model selection was based on BRUVs data from Geographe so will need to be checked and edited for your data.
    - There will be more changes required so careful reading as you run through the scripts is suggested.
